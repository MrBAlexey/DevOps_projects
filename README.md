Более подробную информацию о проектах и использованных технологиях можно увидеть в репозитриях проектов.

1. Проект: D01_Linux‑1 (Основы UNIX/Linux: Установка и администрирование) \
    Практический проект — пошаговый гид по базовому администрированию Linux на виртуальной машине (Ubuntu 20.04 Server LTS, VirtualBox). 
    Покрыты установка системы, создание пользователя, настройка хоста и сети, SSH, мониторинг дисков, работа с логами, настройка планировщика задач CRON и др.
2. Проект DO2_LinuxNetwork-1 (Основы сетей в UNIX/Linux) \
    Практический проект по настройке сетей Linux на виртуальных машинах (Ubuntu 20.04 Server, VirtualBox). 
    Проект охватывает **адресацию IP‑подсетей, статическую маршрутизацию и тонкую настройку firewall (`iptables`)**, 
    а также работу с **DHCP‑сервером (`isc‑dhcp‑server`), NAT (SNAT и DNAT)** и **SSH‑туннелями (`ssh -L` / `ssh -R`)** между несколькими виртуальными машинами.
